零、废话

效仿王垠博客的一篇文章，我也把我常用的软件列出来，并作一点介绍。此外，我十分赞同王垠的观点：科学工作者应该使用Linux作为其操作系统。

一、操作系统

我所有的工作都是基于GNU/Linux的。原因只有一个：效率高。

  • Ubuntu

    新手第一次使用Linux的话，推荐使用Ubuntu入门。配置和上手都比较方便。

  • Arch Linux

    如果希望对Linux系统有更深入的理解，请尝试折腾Arch Linux。

  • FreeBSD

    我没试过FreeBSD。不过鉴于很多服务器都是使用FreeBSD且十分稳定，我很想试试。貌似其安装配置过程比Arch Linux更折腾，应该是系统管理员必修的内容。

二、编辑器

程序员大部分都是在进行编辑操作，所以有个得心应手的编辑器是必须的。

  • Emacs

    其实我的大部分工作并非基于Linux操作系统，而是基于Emacs。Emacs很有趣。

  • Vim

    作为Emacs的宿敌，Vi更轻，而且是系统自带。很多程序员因其高效的编辑效率而选择它，我偶尔也使用Vi。

  • Eclipse

    我只有在偶尔写Java的时候才使用Eclipse。
